---
description: Launch local dev server and spawn browser subagent to test and view the application on localhost
---

1. Start the local development server in the background using `npm run dev` if it is not already running.
2. Launch `browser_subagent` (or `invoke_subagent`) to navigate to `http://localhost:5173`, wait for the app to load, inspect the UI, interact with elements, and verify mobile/desktop responsiveness.
3. Report the observations and capture any relevant screenshots or video recordings.
